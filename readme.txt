== Description ==

WP Magick Toolbox（魔法工具箱）是一款面向中国 WordPress 站长的实用工具箱插件。3.0.0 以 57 个注册模块为运行边界，通过七个语义化管理视图提供站点设置、内容与 SEO、登录安全、国内生态和维护诊断等能力。

= 当前能力 =

* 七个后台视图：概览、站点与媒体、内容与页面、SEO 与增强、国内生态、维护工具、关于与帮助
* 站点与媒体：链接、上传、图片、后台列表和国内 CDN 适配
* 内容与页面：评论治理、隐藏内容、阅读体验和内容维护
* SEO 与增强：TDK、内链、搜索健康和发文统计
* 国内生态与安全：备案展示、微信 JSSDK、Cookie 提示、登录失败限制和匿名作者枚举限制
* 维护工具：站点诊断、SEO 检查、媒体体检和数据库清理；高风险操作需要预览或确认
* 管理体验：功能搜索、风险标签、设置差异确认、敏感凭据状态管理和移动端布局

= 适用场景 =

- 个人博客美化与优化
- 企业网站基础设置与备案合规
- 内容站 SEO 提升

== Installation ==

1. 上传 `wp-magick-toolbox` 文件夹到 `/wp-content/plugins/` 目录
2. 在 WordPress 后台的“插件”菜单中激活该插件
3. 进入“插件 > 魔法工具箱”进行设置

== Frequently Asked Questions ==

= 是否支持多站点？ =

当前版本主要针对单站点优化，多站点支持正在开发中。

= 是否兼容所有主题？ =

不承诺兼容所有主题。依赖主题结构的能力会在后台标注，建议先在测试环境启用并核对页面表现。

= 如何卸载？ =

停用并删除插件即可。卸载时会自动清理插件创建的选项数据。

= 是否支持国际化？ =

支持。插件文本域为 `magick-toolbox`，可通过标准 WordPress 翻译流程进行本地化。

== Privacy Policy ==

启用搜索健康后，插件可能在站点本地数据库记录搜索词、热词计数和无结果计数。启用相关能力后，登录安全、审计与诊断可能在本地记录登录失败、IP 地址、操作事件和诊断结果；站点管理员应按自身隐私政策和保留周期管理这些数据。

插件不会自动把上述本地统计上传给作者。第三方集成只在管理员显式启用并配置后发起请求；国内访问连通性检测只在管理员主动运行检查时请求目标服务。传输内容及后续处理受所选服务和站点配置约束。
