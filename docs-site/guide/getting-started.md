# 安装与使用

## 环境要求

- WordPress 4.6 或更高版本
- PHP 7.4 或更高版本
- Chrome、Firefox、Safari 或 Edge 等现代浏览器

## 安装

1. 下载插件 ZIP 包。
2. 在 WordPress 后台进入 **插件 → 安装插件 → 上传插件**。
3. 上传、安装并启用插件。
4. 在左侧菜单打开 **魔法工具箱**。

也可以解压后将 `wp-magick-toolbox` 目录上传到 `wp-content/plugins/`，再从插件页启用。新安装不需要先执行一次空保存。

## 管理后台

管理后台按任务分为概览、站点与媒体、内容与 SEO、安全、国内生态、维护工具及帮助等区域。URL 中的 `view` 参数可用于收藏或直接打开指定区域。

概览页只展示真实诊断结果和可执行下一步。接口不可用时会明确显示失败状态，不会用默认分数代替真实数据。

## 保存设置

1. 修改需要的功能或参数。
2. 点击固定保存按钮。
3. 查看变更摘要并确认。
4. 等待成功提示后再离开页面。

设置首次读取失败时，保存按钮会保持禁用，避免用本地默认值覆盖服务端数据。

## 密钥字段

微信 AppSecret 和 OSS Access Key/Secret Key 采用独立凭据控件：

- 页面只显示“已配置”或“未配置”，不会回填原值。
- 不操作表示保留已有值。
- 选择替换后输入新值并保存，才会覆盖旧值。
- 选择清除并保存，才会删除服务端值。

这些原值不会出现在页面启动数据、设置读取响应、变更摘要或浏览器快照中。

## 出现异常时

先关闭刚刚启用的功能并保存；若后台无法打开，可在 WordPress 插件页停用插件，然后检查 PHP 错误日志。Pre-GA 版本不提供设置 JSON 导入导出或浏览器快照恢复。
