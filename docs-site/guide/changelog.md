## 3.0.0 — 2026-07-16

### 重构
- 以 Pre-GA clean break 重建管理后台：七个语义化视图承载 57 个注册模块，不保留旧数字导航兼容入口
- 模块 Registry 与配置 Schema 成为单一事实源；前端设置类型、敏感路径和 33 条功能搜索索引由 PHP 契约生成
- 敏感设置只返回配置状态，凭据写入明确区分保留、替换和清除；保存前展示真实差异与高风险确认
- 管理 REST 请求统一经过一个客户端，加载、错误、空状态、键盘路径和移动端反馈采用同一交互边界
- 前端收敛为一个 pnpm 前端工程，以及按页面加载的 Admin、Count 两个独立产物

### 删除
- 删除 AI Provider Runtime 正式产品代码；移植参考只保存在不进入发布包的只读目录
- 删除不可信登录验证码、anti-crawler/腾讯防水墙遗留、百度推送及其设置、REST 和前端入口
- 删除旧 Dashboard 默认分、首次向导、预设市场、收藏拖拽、浏览器设置快照和旧迁移/备份兼容层
- 删除没有可用管理界面或真实消费者的文章批量替换模块及 3 条手动 REST 路由
- 删除数据库表导出、文章评分和微信解锁遗留的 4 条不可调用 REST 路由

### 修复
- 登录失败限制、可信代理 IP、匿名作者枚举保护和锁定恢复路径按独立设置正确加载
- 模块、设置、生成式前端契约和发布 ZIP 增加精确一致性门禁
- 分类、标签和页面选项接口改用受管理员权限保护的标准 WordPress REST 响应
- REST Registry 增加 callback 可调用性与精确产品表面回归门禁
