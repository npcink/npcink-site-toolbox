# 常见问题

## 安装与使用

### 插件支持哪些 WordPress 版本？

WordPress 4.6 及以上版本均可使用。推荐使用最新稳定版以获得最佳兼容性。

### 支持哪些 PHP 版本？

PHP 7.4 ~ 8.3。CI 矩阵已覆盖全部版本的自动化测试。

### 启用后会影响站点性能吗？

不会。插件采用模块注册表 + 按需加载机制：
- 未启用的功能不注册任何 Hook
- 后台资源仅在插件设置页面加载
- 前台资源严格按功能开关和页面条件加载

## 功能相关

### 如何找到特定功能？

设置页面顶部有全局搜索框，支持中文搜索。输入关键词后实时过滤功能卡片，点击结果自动跳转到对应分类。

### 哪些功能建议开启？

进入仪表盘查看「建议开启」列表。一般推荐：
- 移除 WP 版本号
- 搜索频次限制
- 图片 Alt 自动补全
- 首页 TDK
- 登录验证码

### 开启某个功能后站点异常怎么办？

1. 进入仪表盘，查看健康评分变化
2. 使用配置备份恢复到之前的快照
3. 关闭该功能后观察站点是否恢复正常
4. 如问题持续，通过反馈 Tab 提交问题描述

## 配置管理

### 配置数据存储在何处？

按模块拆分为多个 `wp_options` 键：
- `Magick_ToolBox_Option_Core` — 核心设置
- `Magick_ToolBox_Option_SEO` — SEO 功能
- `Magick_ToolBox_Option_Page` — 页面功能
- `Magick_ToolBox_Option_Media` — 媒体优化
- `Magick_ToolBox_Option_Comment` — 评论功能
- `Magick_ToolBox_Option_Security` — 登录安全
- `Magick_ToolBox_Option_Appearance` — 外观特效

### 升级会丢失配置吗？

不会。升级时自动保留所有配置。配置拆分时已实现迁移脚本，老用户升级自动迁移并备份。

### 如何迁移到另一台服务器？

1. 在源站点导出配置（配置备份 → 导出）
2. 在目标站点安装并启用插件
3. 导入配置文件（配置备份 → 导入）

## 安全相关

### 登录失败锁定会影响管理员吗？

不会。锁定机制仅针对登录失败的 IP，管理员从已知设备登录不受影响。

## 开发相关

### 如何添加新功能？

详见 [开发指南](/guide/development)。核心步骤：
1. 在 `admin/partials/[category]/` 创建 PHP 文件
2. 在 `admin/modules/registry.php` 注册模块
3. 在前端类型定义和默认值中添加字段
4. 在对应 Tab 组件中添加 UI 控件

### 是否支持二次开发？

支持。插件基于 GPL-2.0 开源协议，可自由修改和分发。

## 兼容性问题

### 与 SEO 插件冲突怎么办？

如已安装 Yoast SEO、Rank Math 等插件，建议：
- 关闭本插件的首页 TDK、文章 SEO 功能
- 保留本插件的分类 SEO、标签 SEO（如其他插件不支持）
- 避免重复设置导致冲突

### 与缓存插件兼容吗？

兼容。但注意：
- 修改配置后需清除缓存
- 维护页功能可能与缓存冲突
- 评论相关功能需排除评论页面缓存

## 其他问题

### 插件有付费版本吗？

没有。全部功能免费开放，无付费墙或功能限制。

### 如何获取技术支持？

1. 查看本文档和 [功能文档](/features/overview)
2. 访问 [Npcink 官网](https://www.npc.ink) 寻求帮助

### 如何卸载插件？

1. 进入 WordPress 后台 → 插件
2. 停用 WP Magick Toolbox
3. 删除插件
4. 卸载时自动清理所有配置数据（可选保留）
