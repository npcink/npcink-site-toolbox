# 推荐方案

WP Magick Toolbox 内置了多种一键配置方案，帮助你根据站点类型快速套用最佳实践配置。

## 使用方式

1. 进入后台首页（仪表盘）
2. 在**一键配置方案**卡片中选择适合你的方案
3. 点击**应用此配置**
4. 系统会弹出确认对话框，提示此操作将覆盖当前部分配置
5. 确认后自动保存到服务器

::: tip 建议
应用预设前建议先导出当前配置备份，或确认已有自动快照，以便必要时回滚。
:::

## 内置方案

### 个人博客稳定方案

**适用场景**：个人博客、技术博客、生活记录站点

**配置要点**：
- SEO 基础：开启首页/文章/分类 TDK
- 媒体优化：图片 Alt 自动补全、上传文件重命名
- 登录安全：启用数学验证码
- 评论防护：限制评论间隔、字数控制
- 关闭重型特效：背景特效、粒子效果、全站置灰、灯笼等

**核心目标**：稳定、安全、SEO 友好，避免花哨功能拖慢速度。

### 企业官网合规方案

**适用场景**：企业官网、品牌展示站点、政府/机构网站

**配置要点**：
- 合规：开启备案与 Cookie 合规提示
- 安全：登录验证码、隐藏 WP 版本号、邮件 IP 隐藏
- SEO：首页/文章/分类 TDK
- 品牌：登录页美化、自定义登录链接
- 统计：百度统计占位（需手动填写 ID）
- 关闭评论表情包等娱乐化功能

**核心目标**：合规、专业、安全，符合企业形象。

### 内容站 SEO 安全方案

**适用场景**：内容聚合站、资讯站、自媒体站点

**配置要点**：
- 全面 SEO：首页/文章/分类/标签 TDK
- 媒体体检：SEO 检查、媒体库体检
- 搜索治理：限制搜索频次，防止恶意爬虫
- 评论治理：敏感词过滤、百度内容审核、评论间隔限制
- 内容互动：文章评分、分享按钮
- 登录安全：验证码防护

**核心目标**：SEO 最大化、内容安全、搜索友好。

### 极速纯净版

**适用场景**：追求极致性能、服务器资源有限的站点

**配置要点**：
- 仅保留核心站点优化和安全防护
- 关闭所有视觉特效和非必要功能
- CDN 替换、Gravatar 镜像优化

### 性能优先

**适用场景**：高流量站点、电商站点、对加载速度敏感的场景

**配置要点**：
- 关闭所有非必要功能
- 禁用缩略图生成（需主题支持）
- 图片 Alt、上传重命名保留
- 所有特效显式关闭

### 安全防护优先

**适用场景**：频繁遭受攻击、对安全性要求极高的站点

**配置要点**：
- 移除 WP 版本号、站点地图用户信息
- 评论全面防护：间隔、字数、英文过滤、敏感词、百度审核
- 搜索限制、反爬虫
- 禁止微信/QQ 直接打开
- 登录验证码、错误信息替换

## 自定义方案

如果内置方案不能满足需求，你可以保存当前配置为自定义方案：

1. 调整好所有配置项
2. 点击**保存当前为方案**
3. 输入方案名称和描述
4. 保存后，你的自定义方案会出现在方案列表中
5. 可以随时应用或删除自定义方案

::: warning 注意
自定义方案仅保存在当前浏览器的 localStorage 中，换浏览器或清除数据后会丢失。重要配置请使用**导出配置**功能备份为 JSON 文件。
:::

## 第三方凭证

预设方案中涉及第三方服务的字段（如百度统计 ID、Google Analytics ID、微信/百度推送 Token 等）均以占位符形式存在，**不会自动填入真实凭证**。应用预设后，请前往对应模块手动填写你的实际密钥和 ID。
