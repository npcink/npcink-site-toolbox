## 技术架构

- **后端**：PHP 7.4+，WordPress 插件标准架构
- **前端设置页**：React + TypeScript + Vite + Ant Design
- **数据存储**：WordPress `wp_options` 表
- **通信方式**：WordPress REST API

## 安全加固（已完成）

- SQL 注入防护（全部使用 `$wpdb->prepare()`）
- CSRF 防护（nonce 验证）
- XSS 防护（输出转义）
- 权限检查（`current_user_can`）
- 输入清洗（`sanitize_text_field`）
